Third project:HR ATTRITION presentation Ce dashboard offre une vue d’ensemble de la structure de l'netreprise, permettant d’analyser l'attrition, Il est structuré en quatre sections principales : Overview, demographic,attrition et attrition-view. Ce dashboard fournit une analyse détaillé grâce à la présentation de l'etat actuel de l'entreprise par demographie ainsi qu'une representation de l'attrition sur les différents paramètres:

-les départements et métiers
-les années
-la fréquence de voyage des employés
-les heures supplémentaires
-les années vécu au sein de l'entreprise

Dans la partie developper par python :
-On cherche la correlation existante entre l'attrition et les differents paramètres
-On distingue 4 paramètres avec leur taux d'influence sur l'attrition:
satisfaction métier - satisfaction d'environnement - équilibre entre vie privé et vie professionnels -  satisfaction relationnel
